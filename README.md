## Installation
```bash
$ brew install mactex
$ brew install texshop # For GUI interface
# $ pre-commit install
```
Now manually check whether `chktex` and `latexindent` work. If not install the missing packages using `cpan install [PACKAGE]`

## Linting
```bash
$ make lint
```

## Compilation
Simply run make
```bash
$ make
```
